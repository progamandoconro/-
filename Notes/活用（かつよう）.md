# 食べる 🍔：
```
*　食べられる。
*　食べられない。
*　食べられます。
*　食べられました。
*　食べられなった。
*　食べられませんでした。

*　食べることができます。
*　食べることができました。
*　食べることができませんでした。

*　食べたい。
*　食べたくない。
*　食べたいでわありません。

*　お腹が空きました。
```
# 飲める　🍺:
```
*　飲められる。
*　飲められない。
*　飲められます。
*　飲められました。
*　飲められなった。
*　飲められませんでした。

*　飲めたい。
*　飲むたくない。

*　喉が渇いています。　（のどがかわいています）。

```
