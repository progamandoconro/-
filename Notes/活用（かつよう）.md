# 食べる（たべる）🍔：
```
*　食べます。
*　食べて。
*　食べましょ。
*　食べました。
*　食べませんでした。
*　食べなかった。

*　食べられる。
*　食べられない。
*　食べられます。
*　食べられました。
*　食べられなった。
*　食べられませんでした。

*　食べることができます。
*　食べることができました。
*　食べることができませんでした。

*　食べたい。
*　食べたくない。
*　食べたいでわありません。

*　お腹が空きました。
*　いただきます。
*　ごちそうさまでした。
```
# 飲む（のむ）🍺：
```
*　飲められる。
*　飲められない。
*　飲められます。
*　飲められました。
*　飲められなった。
*　飲められませんでした。

*　飲めたい。
*　飲むたくない。

*　喉が渇いています。　（のどがかわいています）。

```
# 歩く（あるく）🚶：
```
*　歩くれる。
*　歩くれるない。
*　歩くできます。
*　歩くできません。
*　歩くできました。
*　歩くできませんでした。

```
# 働く（はたらく）　💼：
```
*　働きます。
*　働きました。
*　働いた。
*　働きましょ。
*　働きたでした。

*　働きたい。
*　働きたかった。

*　お仕事。
*　お金。
*　お疲れ様でした（おつかれさまでした）。

```
