## Programa para el estudio de Kanji Japonés (日本語漢字学習プログラム)

     git clone https://github.com/progamandoconro/Kanji-con-Ro

### Usar con golang:     

     cd アプリ/大き     
     go run kangi.go

### Usar con Docker:

     docker build . -t kanji
     docker run -it kanji
     go run kangi.go
