## Programa para el estudio de Kanji Japonés (日本語漢字学習プログラム)

### Usar con golang:     
     
     go run kangi.go

### Usar con Docker:

     docker build . -t kanji
     docker run -it kanji

### Ejecutar el programa en el contenedor:

     go run kangi.go
